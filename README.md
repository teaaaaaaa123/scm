# SCM 统一查询系统 - API版本

## 功能介绍

本系统用于查询服装订单的生产进度信息，支持三种查询方式：
- **生产单号查询**：通过生产单号（如 *202608066）查询订单
- **客户姓名查询**：通过客户姓名查询订单列表
- **流水号查询**：通过流水号查询订单详情

## 使用方法

### 基本格式
```bash
python scm_query_api.py <查询类型> <查询参数>
```

### 查询类型说明

| 查询类型 | 命令格式 | 示例 |
|---------|---------|------|
| 生产单号 | `订单 <生产单号>` | `python scm_query_api.py 订单 *202608066` |
| 客户姓名 | `客户 <姓名> 订单` | `python scm_query_api.py 客户 蕾蕾（员工） 订单` |
| 流水号 | `流水号 <流水号>` | `python scm_query_api.py 流水号 11374` |

## 示例

### 1. 生产单号查询
```bash
python scm_query_api.py 订单 *202608066
```

### 2. 客户姓名查询
```bash
python scm_query_api.py 客户 刘浩（员工） 订单
```

### 3. 流水号查询
```bash
python scm_query_api.py 流水号 11372
```

## 查询结果说明

查询结果包含以下信息：
- **订单信息**：订单号、订单ID
- **明细信息**：流水号、版型、面料、颜色、尺码、数量
- **量体信息**：胸围、衣长、肩宽等（根据版型不同）
- **生产进度**：发料、前道、中道、后道、入库、出库状态

## 注意事项

1. 生产单号通常以 `*` 开头，如 `*202607578`
2. 客户姓名支持模糊匹配，建议使用完整姓名以获得准确结果
3. 流水号为纯数字，如 `11374`
4. 查询需要网络连接，首次查询可能需要稍等获取认证令牌
5. 如果生产进度显示"未获取到生产进度"，可能是订单还未进入生产环节

## 系统要求

- Python 3.6+
- requests 库（可通过 `pip install requests` 安装）

## 运行脚本

也可以直接双击运行 `run.bat` 文件，按提示输入查询内容。
